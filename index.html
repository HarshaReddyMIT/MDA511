Week 1

Functions & Models

Lecture
What is a Mathematical Model?

A mathematical description (function or equation) of a real-world phenomenon. Models simplify reality enough to permit calculations but must be accurate enough to be useful. They are always idealisations — "Mother Nature has the final say."

The modelling process: Real-world problem → Formulate model → Solve mathematically → Interpret results → Test against reality

Four Ways to Represent a Function
  • Verbally: describe in words (e.g. "temperature depends on time")
  • Numerically: table of values
  • Visually: graph
  • Algebraically: formula, e.g. A = πr²
Domain & Range
  • Domain = set of valid inputs x
  • Range = set of all output values f(x)
  • Independent variable = x (input)
  • Dependent variable = y = f(x) (output)
Example: f(x) = √x. Domain: [0, ∞). Range: [0, ∞).
Vertical Line Test

A curve in the xy-plane is the graph of a function if and only if no vertical line intersects the curve more than once.

x = y² fails the test — vertical lines cross it twice. Its two halves y = √(x+2) and y = −√(x+2) ARE functions.
Even & Odd Functions
Even: f(−x) = f(x) → symmetric about y-axis
Odd: f(−x) = −f(x) → symmetric about origin
  • f(x) = x⁵ + x → odd (check: f(−x) = −x⁵ − x = −f(x) ✓)
  • g(x) = 1 − x⁴ → even (check: g(−x) = 1 − x⁴ = g(x) ✓)
  • h(x) = 2x − x² → neither
Increasing & Decreasing

f is increasing on interval I if x₁ < x₂ ⟹ f(x₁) < f(x₂)

f is decreasing on interval I if x₁ < x₂ ⟹ f(x₁) > f(x₂)

f(x) = x² is decreasing on (−∞, 0] and increasing on [0, ∞).
Piecewise Defined Functions

Different formulas on different parts of the domain. Example:

f(x) = { 1 − x if x ≤ −1
{ x² if x > −1

Evaluate: f(−2) = 1−(−2) = 3; f(−1) = 1−(−1) = 2; f(0) = 0² = 0

Step functions (like postage cost) are piecewise — they jump between constant values.

Linear Models in Detail
y = f(x) = mx + b (m = slope = rate of change, b = y-intercept)

Example: Temperature vs altitude: Ground temp 20°C, at 1km = 10°C.

Slope m = (10 − 20)/(1 − 0) = −10°C/km. So: T = −10h + 20

At h = 2.5 km: T = −10(2.5) + 20 = −5°C

The slope represents the rate of change — here, temperature drops 10°C per km of altitude.
Empirical Models & Regression
  • Empirical model: built entirely from data when no physical law applies
  • Scatter plot: plot data points (x, y) to visually assess the relationship
  • Regression line: best-fit line that minimises sum of squared vertical distances (least squares)
  • Y-intercept of regression line: predicted y when x = 0 — interpret carefully in context
  • Extrapolation: predicting beyond data range — risky but sometimes asked for
Tutorial Q: Asbestos exposure (fibres/mL) vs % lung cancer. Plot scatter → assess linearity → find regression equation → estimate % at 4000 fibres/mL by substituting x = 4000 into the regression equation.

Week 2

Limits & Derivatives

Lecture
The Tangent Problem — How Derivatives Arise

To find the slope of a tangent line at point P, we use nearby point Q and compute the secant line slope, then take the limit as Q → P.

Secant slope: m_PQ = [f(x) − f(a)] / (x − a)
Tangent slope: m = lim_{x→a} [f(x) − f(a)] / (x − a)

Example: y = x² at P(1,1). Choose Q(1.5, 2.25): m_PQ = (2.25−1)/(1.5−1) = 2.5. As x→1, m_PQ → 2. So tangent is y − 1 = 2(x − 1) → y = 2x − 1

Limits — Definition & Theorem
lim_{x→a} f(x) = L

Means f(x) gets arbitrarily close to L as x approaches a (but x ≠ a).

Two-sided limit exists ⟺ lim_{x→a⁻} f(x) = L = lim_{x→a⁺} f(x)
lim_{x→0} |x|/x does NOT exist: left limit = −1, right limit = +1. They differ!
lim_{x→0} |x| = 0: left limit = 0, right limit = 0. They match — limit exists!
Derivative — The Core Definition
f′(a) = lim_{h→0} [f(a+h) − f(a)] / h
f′(x) = lim_{h→0} [f(x+h) − f(x)] / h (as a function)
  • f′(x) = slope of the tangent line at any point x
  • f′(x) = instantaneous rate of change of f at x
  • The domain of f′ may be smaller than the domain of f
Derivative Notation — All Forms
f′(x) = y′ = dy/dx = df/dx = d/dx[f(x)] = Df(x)

At a specific point x = a:

f′(a) = dy/dx|_{x=a}

Leibniz notation dy/dx is NOT a ratio — it is the limit of Δy/Δx.

Velocity Problem — Average vs Instantaneous
Average velocity = Δs/Δt = [s(t₂) − s(t₁)] / (t₂ − t₁)
Instantaneous velocity = v(t) = s′(t) = lim_{Δt→0} Δs/Δt

Example: s(t) = 4.9t² (free fall). Average velocity from t=5 to t=5.1:

[s(5.1) − s(5)] / 0.1 = [4.9(5.1)² − 4.9(25)] / 0.1 ≈ 49.49 m/s

As the interval shrinks → instantaneous velocity at t=5 = 49 m/s

Higher Derivatives
f″(x) = (f′)′ = d²y/dx² (second derivative)
f‴(x) = (f″)′ (third derivative)
f⁽ⁿ⁾(x) (nth derivative)

s(t)Position

Three Ways a Function Fails to Be Differentiable

1. Corner/Kink

Left & right derivatives exist but differ. Example: f(x) = |x| at x=0

2. Discontinuity

Any jump, removable, or infinite discontinuity. Must be continuous to be differentiable.

3. Vertical Tangent

Slope approaches ±∞. The limit of the difference quotient does not exist (finite).

Key theorem: Differentiable at a ⟹ Continuous at a. The CONVERSE IS FALSE — |x| is continuous everywhere but not differentiable at 0.
Example: Differentiating f(x) = |x|

For x > 0: f′(x) = 1. For x < 0: f′(x) = −1. At x = 0: left limit of difference quotient = −1, right = +1. Since −1 ≠ 1, f′(0) does not exist.

f′(x) = { 1 if x > 0 } { −1 if x < 0 }

Week 3

Differential Calculus & Optimisation

Lecture
Absolute vs Local Extrema
  • Absolute (global) max: f(c) ≥ f(x) for ALL x in domain
  • Absolute (global) min: f(c) ≤ f(x) for ALL x in domain
  • Local (relative) max: f(c) ≥ f(x) for x near c
  • Local (relative) min: f(c) ≤ f(x) for x near c
  • Local extrema occur only in the interior, never at endpoints
  • A local extremum CAN also be an absolute extremum
Extreme Value Theorem

If f is continuous on a closed interval [a, b], then f attains both an absolute maximum AND an absolute minimum on [a, b].

BOTH conditions required:
Without continuity → function may "skip" its extreme value
Without closed interval → function may approach but never reach the extreme
Fermat's Theorem & Critical Numbers

Fermat's Theorem: If f has a local max or min at c, and f′(c) exists, then f′(c) = 0.

The CONVERSE IS FALSE: f′(c) = 0 does NOT guarantee an extremum. Example: f(x) = x³ at c=0 — f′(0) = 0 but the curve just crosses its horizontal tangent.
Critical number c: f′(c) = 0 OR f′(c) does not exist

Example: f(x) = |x| has a local min at x=0, and f′(0) does not exist — still a critical number.

Closed Interval Method — Find Absolute Extrema
  1. Find all critical numbers in the open interval (a, b): solve f′(x) = 0 AND find where f′(x) DNE
  2. Evaluate f at each critical number AND at the endpoints x = a and x = b
  3. Compare all values: largest = absolute max; smallest = absolute min
This method ALWAYS works for a continuous function on a closed interval.
Optimisation Problem Steps
  1. Understand: What quantity Q are you maximising/minimising?
  2. Diagram: draw and label all variables
  3. Notation: assign Q and other variables symbols
  4. Express Q as a function of one variable (use constraints to eliminate others)
  5. State domain of the function in context
  6. Apply Closed Interval Method to find absolute max/min
Farmer Fence — Full Worked Solution

2400 ft of fencing, rectangular field along river (no fence on river side). Maximise area.

Let x = depth (perpendicular to river), y = width (parallel to river)
Constraint: 2x + y = 2400 → y = 2400 − 2x
A(x) = xy = x(2400 − 2x) = 2400x − 2x², 0 ≤ x ≤ 1200
A′(x) = 2400 − 4x = 0 → x = 600

Check values: A(0) = 0, A(600) = 2400(600) − 2(600²) = 720,000 ft², A(1200) = 0

Maximum area = 720,000 ft² at x = 600 ft, y = 1200 ft. Field is 600 ft × 1200 ft.

Business Applications
Cost: C(x) = cx (c = cost per unit, x = quantity)
Marginal cost: C′(x) (cost of one additional unit)
Revenue: R(x) = x · p(x) (p(x) = price function)
Marginal revenue: R′(x)
Profit: P(x) = R(x) − C(x)
Marginal profit: P′(x) = R′(x) − C′(x)

Profit is maximised when R′(x) = C′(x) (marginal revenue = marginal cost)

TV Rebate — Full Worked Solution

Currently sells 200 TVs/week at $350. Each $10 rebate → 20 more sold. Find rebate to maximise revenue.

x = TVs sold per week. Extra TVs = x − 200. Each extra 20 TVs → $10 price drop.
Price per TV drop = (x − 200)/20 × $10 = (x−200)/2
p(x) = 350 − (x−200)/2 = 450 − x/2
R(x) = xp(x) = 450x − x²/2
R′(x) = 450 − x = 0 → x = 450 TVs
p(450) = 450 − 450/2 = 225

Rebate = 350 − 225 = $125. Revenue = 450 × 225 = $101,250/week

PBL Week 4 — Boyle's Law & Quotient Rule

Boyle's Law: PV = C (constant at constant temperature).

V = C/P = C · P⁻¹
dV/dP = −C/P²

At P = 50 kPa, V = 0.106 m³ → C = PV = 50 × 0.106 = 5.3 kPa·m³

dV/dP = −5.3/50² = −5.3/2500 = −0.00212 m³/kPa

Meaning: At P=50 kPa, each additional 1 kPa of pressure reduces volume by 0.00212 m³.

Quotient Rule — Detail
If R(x) = f(x)/g(x), then:
R′(x) = [f′(x)·g(x) − f(x)·g′(x)] / [g(x)]²

Mnemonic: "Low d-High minus High d-Low, over Low squared"

PBL Q: For R(x) = (x − 3x³ + 5x⁵)/(1 + 3x³ + 6x⁶ + 9x⁹), find R′(0). Hint: let f = numerator, g = denominator. At x=0: f(0)=0, g(0)=1, f′(0)=1, g′(0)=0. R′(0) = (1·1 − 0·0)/1² = 1

Chain Rule — Related Rates
d/dt[V(r(t))] = dV/dr · dr/dt

Balloon problem: V = (4/3)πr³

dV/dr = 4πr²
dV/dt = 4πr² · dr/dt

dV/dr: rate of change of volume with respect to radius (units: m³/m = m²)

dV/dt: rate of change of volume with respect to time (units: m³/s)

dr/dt: rate at which radius is growing (units: m/s)

Boyle's Law — Full Solution

PV = C at constant temperature. At P=50 kPa, V=0.106 m³ → C = 5.3 kPa·m³

V(P) = C/P = 5.3/P
dV/dP = −C/P² = −5.3/P²
At P = 50: dV/dP = −5.3/2500 = −0.00212 m³/kPa

Meaning: At P=50 kPa, each 1 kPa increase in pressure decreases volume by 0.00212 m³. Negative because pressure and volume are inversely related.

Net Change Theorem

The integral of a rate of change equals the net change:

∫ₐᵇ F′(x) dx = F(b) − F(a)

This is the fundamental connection between differentiation and integration. If you know the rate of change (derivative), integrate it to get the total change.

y can increase, then decrease, then increase again — F(b) − F(a) gives the NET change, not total distance travelled.
Riemann Sums — Approximating Definite Integrals

Approximate the area under a curve using rectangles:

Left endpoint: Σ f(xᵢ₋₁)·Δx
Right endpoint: Σ f(xᵢ)·Δx

As the number of rectangles → ∞, the Riemann sum → the exact definite integral.

Bernhard Riemann (1826–1866): formalised the concept of the integral as the limit of sums. His work underpins all of modern integration theory and real analysis.
Displacement vs Distance — Critical Distinction
Displacement = ∫ₐᵇ v(t) dt (signed — can be positive, negative, or zero)
Distance = ∫ₐᵇ |v(t)| dt (always ≥ 0)

When v(t) changes sign in [a,b], split the integral at each zero:

Find t* where v(t*) = 0 in (a,b)
Distance = |∫ₐᵗ* v(t) dt| + |∫ₜ*ᵇ v(t) dt|
If you compute ∫v(t)dt without taking |v|, you get displacement only. Positive and negative parts CANCEL — you'll undercount distance.
v(t) = t² − 2t − 3 on [2, 4] — Full Worked Solution

Step 1: Find zeros of v in [2,4]

t² − 2t − 3 = (t−3)(t+1) = 0 → t = 3 (in [2,4]) or t = −1 (outside)

v < 0 on [2, 3] (particle moves backward). v > 0 on [3, 4] (particle moves forward).

Step 2: Antiderivative

∫(t² − 2t − 3) dt = t³/3 − t² − 3t + C

Step 3: Displacement

∫₂⁴ v dt = [t³/3 − t² − 3t]₂⁴
= (64/3 − 16 − 12) − (8/3 − 4 − 6)
= (64/3 − 28) − (8/3 − 10)
= (64/3 − 84/3) − (8/3 − 30/3)
= −20/3 − (−22/3) = 2/3 metres

Step 4: Distance

|∫₂³ v dt| = |[t³/3 − t² − 3t]₂³| = |(9−9−9)−(8/3−4−6)| = |−9−(−22/3)| = |−5/3| = 5/3
∫₃⁴ v dt = [t³/3 − t² − 3t]₃⁴ = (64/3−16−12)−(9−9−9) = −20/3−(−9) = 7/3
Total distance = 5/3 + 7/3 = 12/3 = 4 metres

Q2 (Week 1) — Determine whether h(x) = 2x − x² is even, odd, or neither. W1

Test by computing h(−x): h(−x) = 2(−x) − (−x)² = −2x − x²

Compare: h(x) = 2x − x². Is h(−x) = h(x)? → −2x − x² ≠ 2x − x². Not even.

Is h(−x) = −h(x)? → −h(x) = −2x + x². Is −2x − x² = −2x + x²? No. Not odd.

Q3 (Week 1 Lab) — What is the difference between / and // in Python? What does int(9.8) return and why? W1

/ is true (float) division: 3/2 = 1.5. Always returns a float even if the result is a whole number (e.g. 4/2 = 2.0).

// is floor (integer) division: 3//2 = 1. Always rounds DOWN to the nearest integer. Note: −3//2 = −2 (rounds DOWN, not toward zero).

int(9.8) returns 9. The int() function TRUNCATES (chops off the decimal part) — it does NOT round. So int(9.4) = 9 and int(9.8) = 9 both, even though 9.8 is closer to 10.

Q4 (Week 2) — Explain why lim_{x→0} |x|/x does not exist. Show using left and right limits. W2

Using the theorem: a two-sided limit exists if and only if left-hand = right-hand limit.

Left-hand limit (x → 0⁻): For x < 0, |x| = −x, so |x|/x = −x/x = −1. Therefore lim_{x→0⁻} |x|/x = −1.

Right-hand limit (x → 0⁺): For x > 0, |x| = x, so |x|/x = x/x = 1. Therefore lim_{x→0⁺} |x|/x = 1.

Since −1 ≠ 1, the two-sided limit does not exist.

Q5 (Week 2) — Three ways a function can fail to be differentiable at a point. Give an example of each. W2

1. Corner/Kink: The left and right derivatives exist but are different. Example: f(x) = |x| at x=0. Left derivative = −1, right derivative = +1. They differ, so f′(0) DNE.

2. Discontinuity: If f is not continuous at a, it cannot be differentiable there (differentiable ⟹ continuous, so not continuous ⟹ not differentiable). Example: any step function at its jump point.

3. Vertical tangent: The derivative limit approaches ±∞. Example: f(x) = x^(1/3) at x=0 — the tangent line is vertical. The slope is undefined (infinite).

Remember: Differentiable ⟹ Continuous. The converse is FALSE — |x| is continuous at 0 but not differentiable there.

Q7 (Week 3) — State the Extreme Value Theorem and explain why BOTH conditions are necessary with counterexamples. W3

Extreme Value Theorem: If f is continuous on a closed interval [a,b], then f attains both an absolute maximum and an absolute minimum on [a,b].

Why continuity is needed: Consider f defined on [0,2] where f(x) = x for x < 1 and f(x) = 0 for x = 1 and f(x) = x for x > 1 (with a removable discontinuity at x=1). The function never actually reaches values near 1 — it skips over them. No maximum is attained in the usual sense.

Why closed interval is needed: On the open interval (0, 2), f(x) = 1/x is continuous but has no maximum — it approaches ∞ as x → 0⁺ but never reaches it. Similarly g(x) = x has no maximum on (0,2) — it gets arbitrarily close to 2 but never reaches it.

Q8 (Week 3) — A store sells 200 flat-screen TVs at $350 each. Each $10 rebate increases sales by 20 TVs/week. Find the rebate that maximises revenue. Show all working. W3

Let x = TVs sold per week. Extra TVs = x − 200. For each 20 extra TVs, price drops $10.

Price decrease per extra TV = $10/20 = $0.50.

Price function: p(x) = 350 − 0.5(x − 200) = 350 − 0.5x + 100 = 450 − 0.5x

Revenue: R(x) = x · p(x) = x(450 − 0.5x) = 450x − 0.5x²

R′(x) = 450 − x. Set R′(x) = 0: x = 450 TVs.

R″(x) = −1 < 0 → confirms this is a maximum (parabola opening downward).

Optimal price: p(450) = 450 − 0.5(450) = 450 − 225 = $225

Rebate = Original price − Optimal price = 350 − 225 = $125

Maximum revenue = 450 × 225 = $101,250 per week

Q12 (Week 5) — What is the Net Change Theorem? How does it connect to the SARS deaths problem and the concept of a Riemann sum? W5

Net Change Theorem: ∫ₐᵇ F′(x) dx = F(b) − F(a). The integral of a rate of change equals the net change in the quantity.

Connection to SARS: Let D(t) = deaths per day (the rate of change of total deaths). Then ∫ D(t) dt = total deaths over that period. The area under the D(t) curve equals the net change in total death count.

Riemann sum approximation: Since we only have data at 14-day intervals, we approximate ∫D(t)dt using rectangles. Each rectangle has width Δt = 14 days and height = D(tᵢ) deaths/day. Area of one rectangle = D(tᵢ) × 14 = deaths in that 14-day interval. Sum of all rectangles ≈ total deaths from March 1 to May 24.

Left endpoint estimate: 14 × (0.0079 + 0.0638 + 0.1944 + 0.4435 + 0.5620 + 0.4630) ≈ 24.3 deaths
